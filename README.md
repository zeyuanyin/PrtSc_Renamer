# PrtSc_Renamer

Win键+PrtSc键：直接生成全屏截图，自动命名为"屏幕截图(1).png"、"屏幕截图(2).png"...

此程序根据图片"修改日期"来重命名为"屏幕截图 YYYY-MM-DD HHMMSS.png"
